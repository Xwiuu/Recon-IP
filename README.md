# ⚡ MISSÃO

> **ReconIP** é uma engine de reconhecimento OSINT de altíssima capacidade — 36 módulos integrados em Shell Script puro. De geolocalização a testes de CVE, de Google Earth a brute-force de subdomínios, tudo isso em uma única TUI (Terminal User Interface) com fallbacks inteligentes e exportação multi-formato.

```
┌─ TARGET ─────────────────────────────────────────────────────┐
│                                                               │
│   🌐  IP/Domain Recon         📍  Geopositioning & CEP       │
│   🔬  Port Scan (TCP/UDP)     🏭  Banner Grabbing           │
│   🧬  DNS Enumeration          🔥  Vulnerability Tests       │
│   🛡️  SSL/TLS Analysis         📧  Email Security            │
│   ☁️  Cloud Detection          📦  CMS Detection             │
│   🌍  Google Earth KML/KMZ     🔁  Reverse IP / DNS         │
│   ───────────────────────────────────────────────────────     │
│   📊  Export: HTML · JSON · CSV · MD · GeoJSON · PDF        │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

```bash

sudo apt update && sudo apt install -y curl jq whois dnsutils mtr-tiny && \
git clone https://github.com/Xwiuu/Recon-IP.git && \
cd Recon-IP && chmod +x reconip.sh super_recon.sh lib/*.sh && \
```
